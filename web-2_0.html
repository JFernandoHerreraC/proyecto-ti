WEB 2.0

La Web 2.0 son sitios web y aplicaciones que utilizan contenido generado por usuarios finales. La Web 2.0 se caracteriza por una mayor interactividad y colaboración del usuario, una conectividad de red más generalizada y canales de comunicación mejorados.

El término Web 2.0 fue acuñado por el consultor de arquitectura de la información Darcy DiNucci en 1999 para diferenciar la burbuja post-puntocom . Posteriormente fue popularizado por O'Reilly Media durante la Conferencia Web 2.0 en 2004. La Web 2.0 refleja la nueva era de Internet, que pone mayor énfasis en las redes sociales , la computación en la nube, mayores niveles de participación y el intercambio de información entre los usuarios de Internet. Si bien la Web 2.0 no significa una mejora técnica, sí refleja un cambio en la forma en que se consume Internet. Los sitios de redes sociales , aplicaciones web y plataformas de autoedición (como Facebook y WordPress ) ganaron popularidad durante este cambio.
